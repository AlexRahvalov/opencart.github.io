Просмотр витрины магазина

Этот гид предназначен для того, чтобы познакомить вас с витриной магазина OpenCart по умолчанию. Витрина магазина демонстрирует, как клиент видит и взаимодействует с магазином.

Тема по умолчанию

OpenCart поставляется с темой по умолчанию после свежей установки:

Товары, представленные выше, включены в качестве образцов данных с установкой OpenCart. Эти товары можно легко удалить и заменить товарами вашего магазина позже.

Этот гид охватывает основы просмотра витрины магазина с темой по умолчанию. Вы можете найти постоянно расширяющийся список доступных тем в Каталоге расширений OpenCart.

Навигация по магазину

Тема по умолчанию OpenCart делает навигацию по товарам магазина легко доступной для его клиентов.

Анатомия главной страницы

Главная страница, безусловно, является самой важной страницей в магазине с точки зрения представления. В большинстве случаев это будет первая страница, с которой взаимодействует клиент (особенно если он попал на сайт магазина из поисковой системы). Главная страница магазина должна быть удобной для пользователя, одновременно подчеркивая товары магазина.

Первый шаг к знакомству с витриной магазина — понимание анатомии главной страницы OpenCart по умолчанию.

Заголовок

Заголовок будет отображаться в верхней части страницы на каждой странице магазина, а не только на главной странице.

В заголовке есть следующие навигационные опции:

  • Логотип магазина: Нажатие на этот логотип перенаправит клиента обратно на главную страницу магазина.
  • Блок валюты: Клиент может выбрать, в какой валюте будут товары магазина, нажав на любые из значков валют.
  • Корзина: Отображает количество купленных товаров и общую стоимость заказа. Нажатие на кнопку откроет все товары, добавленные в корзину, с опциями "Просмотреть корзину" или "Оформить заказ".
  • Поисковая строка: Клиенты могут ввести текст в поисковой строке, чтобы найти продукт в категориях товаров магазина.
  • Ссылки: Ссылки, ведущие на главную страницу, Список желаемого, Мой аккаунт, Корзину и Оформление заказа.
  • Телефон: Номер телефона компании.
  • Мой аккаунт: Клиент может зарегистрироваться или войти в систему отсюда.

Верхнее меню

Верхнее меню категории отображает только верхние родительские категории продуктов. Смотрите Категории, чтобы узнать больше о том, как создавать и назначать категории продуктов.

Как и заголовок, верхнее меню будет отображаться на каждой странице. Когда курсор мыши наводится на категорию, открывается выпадающее меню с подкатегориями для этой родительской категории.

Когда нажимается на родительскую категорию, клиент будет перенаправлен на страницу категории, которая отображает все продукты в этой категории.

Слайд-шоу

Слайд-шоу отображает несколько рекламных баннеров на ваш выбор, чередуя изображения в слайд-шоу. Через определенный промежуток времени один баннер сменяется на следующий. Баннеры в этом слайд-шоу полезны для выделения определенных продуктов, чтобы клиентам было удобно к ним получить доступ. При нажатии на баннер клиент будет перенаправлен на страницу продукта, представленного на баннере.

В отличие от верхнего меню и заголовка, слайд-шоу в стандартной версии OpenCart может быть просмотрено только на главной странице в этом положении.

Рекомендуемые продукты

OpenCart предоставляет возможность выделять определенные продукты на главной странице.

Раздел "Рекомендуемые" включает изображение продукта, его название, цену и возможность добавить продукт прямо в корзину.

Карусель расположена только на главной странице в стандартной версии.

Подвал

Подвал расположен внизу каждой страницы, а не только на главной. Этот блок различных ссылок полезен для сортировки соответствующих страниц для клиента, которые могут не логично вписываться в другие разделы.

Организационная схема подвала может быть разделена на следующие секции:

  • Информация: "О нас", "Информация о доставке", "Политика конфиденциальности", "Условия и положения"
  • Обслуживание клиентов: "Свяжитесь с нами", "Возвраты", "Карта сайта"
  • Дополнения: "Бренды", "Подарочные сертификаты", "Партнеры", "Специальные предложения"
  • Мой аккаунт: "Мой аккаунт", "История заказов", "Список желаемого", "Рассылка новостей"

Страницы продукта

Страница продукта по умолчанию в OpenCart будет следовать структурному формату, представленному ниже.

Страницу продукта можно разделить на следующие секции:

  • Изображение продукта: Изображение продукта может отображаться под названием слева, а также альтернативные виды продукта могут находиться ниже в меньших блоках. Нажатие на основное изображение развернет его в окне, чтобы клиент мог увидеть его более детально.
  • Детали продукта: Код продукта, доступность и цена отображаются справа от изображения продукта.
  • Корзина: Клиент может выбрать количество и добавить продукт в свою корзину, список желаемого или сравнить.
  • Оценка/Поделиться: Под корзиной клиент может оценить продукт и/или поделиться им на различных социальных сетях.
  • Вкладка описания: Область под основной информацией о продукте для предоставления детального описания продукта.
  • Вкладка отзыва: Область для клиента, чтобы написать отзыв о продукте.

Списки продуктов по категориям

Списки продуктов по категориям позволяют клиентам просматривать продукты, схожие с другими продуктами в одной категории. Это особенно полезно для клиентов, желающих сравнить продукты, функция, которая будет объяснена в разделе Категории. Страницу категории можно открыть несколькими способами. Она доступна из верхнего меню, когда клиент нажимает на одну из родительских категорий. Также на страницах продуктов клиент может перейти на страницу списка продуктов категории, нажав на категорию в блоке категорий слева.

Как видно выше, блок категорий отображается слева, как и на странице продукта. Под заголовком категории вверху есть место для добавления описания категории. Ссылка "Уточнить поиск" ведет к подкатегориям данной категории для просмотра. Продукты могут отображаться в зависимости от предпочтений клиента: в списке или сетке. На приведенном выше изображении используется формат списка. Продукты можно сортировать по имени, цене, рейтингу или модели в поле "Сортировать по". Количество отображаемых продуктов в списке можно изменить в поле "Показать" от 15 до 100.

Существует раздел, предоставляющий место для каждого из продуктов в категории, включая изображение продукта, описание, цену и опцию "Добавить в корзину". Есть возможность добавить продукт в список желаемого. Также доступна опция "Добавить для сравнения".

Сравнение продуктов

Функция "Добавить для сравнения" в разделе продуктов позволяет клиенту сравнивать различные спецификации, характеристики и цены нескольких продуктов, которые его интересуют.

Клиенту предоставляется возможность добавить один из сравненных продуктов в корзину, если он этого хочет. Нажатие на кнопку "Продолжить" вернет пользователя на главную страницу.

Страница корзины покупок

После добавления товара в корзину клиент может получить доступ к корзине покупок в заголовке под пунктом "Корзина покупок".

Корзина покупок предоставляет обзор выбранного товара, включая категории "Изображение", "Название продукта", "Модель", "Количество", "Цена за единицу" и "Итого". Клиент может добавить промо-код или подарочный сертификат, а также оценить стоимость доставки и налоги перед переходом к оформлению заказа. Кнопка "Продолжить покупки" возвращает на главную страницу.

Создание аккаунта

Перед тем как клиент сможет продолжить оформление заказа с товара из корзины, ему необходимо выбрать либо оформление заказа как гость, либо войти в свой аккаунт. Оформление заказа как гость не требует ввода данных для входа. Повторные клиенты могут захотеть создать аккаунт в магазине.

Существует несколько способов, как клиент может создать аккаунт:

  1. Оформление заказа

    Шаг 1 процесса оформления заказа позволяет пользователю создать аккаунт перед продолжением оплаты. Выбор "Зарегистрировать аккаунт" изменит Шаг 2 оформления заказа с "Платежные данные" на "Данные аккаунта и платежа". В разделе "Данные аккаунта и платежа" запрашиваются те же личные данные, что и в разделе "Платежные данные", кроме того, запрашивается создание пароля для аккаунта. После завершения Шага 2 клиент может продолжить процесс оформления заказа.

  2. Заголовок - "Мой аккаунт"

    Нажатие на "Мой аккаунт" в заголовке откроет опцию для клиента войти или создать аккаунт. Эта страница предоставляет клиенту возможность войти в систему, если у него уже есть аккаунт, или создать новый аккаунт. В разделе "Новый клиент" клиент может нажать "Продолжить" под "Зарегистрировать аккаунт", чтобы перейти на страницу "Зарегистрировать аккаунт".

Оформление заказа

После добавления товара в корзину клиент может перейти к оформлению заказа для завершения покупки. Страницу оформления заказа можно открыть в заголовке каждой страницы (под полем поиска). Процесс оформления заказа в OpenCart является простым и может быть завершен за 6 шагов.

  1. Шаг 1: Опции оформления заказа

    Клиент может войти в свою учетную запись или зарегистрироваться (как описано выше) или выбрать оформление заказа как гость.

  2. Шаг 2: Платежные данные

    Личные данные, включая "Имя", "Фамилия", "Электронная почта" и "Телефон", заполняются в форме. Также требуется указать адрес клиента.

  3. Шаг 3: Данные доставки

    В разделе платежных данных пользователь может отметить поле, чтобы указать, что данные для доставки совпадают с платежными. Это приведет к пропуску этого шага и переходу к методу доставки. Если данные доставки отличаются от платежных, клиент может ввести эту информацию в форме в этом разделе.

  4. Шаг 4: Метод доставки

    Здесь выбирается способ доставки. Также добавляется поле для комментариев, чтобы клиент мог добавить комментарии к своему заказу.

  5. Шаг 5: Метод оплаты

    Клиент выбирает способ оплаты и может добавить комментарии в соответствующее поле.

  6. Шаг 6: Подтверждение заказа

    На последнем шаге клиент увидит обзор своей покупки, включая описание товара, количество и цену (с учетом налогов и доставки).

Настройка витрины магазина с помощью административной панели

Все перечисленные выше функции можно настроить в административной панели. Администратор может изменять расположение определенных товаров, отключать категории, редактировать цены и описания, загружать баннеры и т.д. В административной панели OpenCart доступно множество возможностей для формирования бренда магазина.
