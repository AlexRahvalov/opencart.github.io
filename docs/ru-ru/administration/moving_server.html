Перемещение OpenCart на новый сервер

OpenCart можно легко перенести на новый веб-сервер. Шаги перечислены ниже:

  1. Скопируйте все файлы OpenCart с вашего существующего веб-сервера на новый веб-сервер.
  2. Экспортируйте базу данных OpenCart и импортируйте её на новый сервер.
  3. Отредактируйте файлы config.php и admin/config.php. Эти файлы содержат пути к файловой системе папок OpenCart, URL-адреса для фронтенда и административной части OpenCart, а также данные для доступа к базе данных. Путь и данные базы данных, скорее всего, нужно будет обновить, а URL-адреса нужно будет изменить только в том случае, если домен магазина OpenCart изменился.
