Фильтр

Фильтр

Фильтр — это удобный инструмент, который регулярно используется в административной части OpenCart. Вы можете ввести информацию в одно из полей для элемента в списке элементов. Нажатие на кнопку "Фильтр" позволит "отфильтровать" все элементы в списке, которые не соответствуют введённым критериям, оставив только те, которые соответствуют. С помощью фильтра вы можете быстро находить любую информацию, хранящуюся в виде списка. Продукты, Категории, Клиенты, Отчеты, Заказы и многое другое используют фильтр в качестве основного инструмента поиска для нахождения элементов в списке.

Например, если вы хотите найти продукт в списке продуктов, вы можете использовать "Фильтр" как ярлык для поиска этого продукта. Это может быть полезно, когда в магазине много продуктов, и вы хотите отредактировать конкретный продукт, не просматривая каждую страницу. Поля поиска расположены в первой строке над списком продуктов.

На скриншоте выше в поле "Название продукта" введено слово "nikon". "Фильтр" отображает только те продукты, которые соответствуют "nikon". Nikon D300 — единственный отображаемый продукт, так как это единственное название продукта, содержащее слово "nikon" в этом магазине. Если слово "nikon" присутствовало бы в названиях нескольких продуктов, кнопка "Фильтр" отобразила бы все из них.
