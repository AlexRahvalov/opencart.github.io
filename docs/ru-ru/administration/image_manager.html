Менеджер изображений

Менеджер изображений — это инструмент, используемый в административной части для загрузки файлов изображений. Баннеры, изображения продуктов, логотип магазина и изображения производителей требуют загрузки с помощью Менеджера изображений.

Эта функция может быть открыта в страницах Категорий, Продуктов, Баннеров, Настроек магазина и других, где необходимо редактировать изображение для магазина.


Нажав на , вы откроете новое окно под названием "Менеджер изображений". Нажмите кнопку "Загрузить" на панели инструментов, чтобы открыть файловую директорию вашего компьютера для непосредственного добавления изображений. Изображения ниже предоставлены для стандартных продуктов. Выберите изображение, нажав на одно из них, и нажмите кнопку "Удалить" на панели инструментов, чтобы удалить его.
