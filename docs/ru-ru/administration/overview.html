Интерфейс администратора

Подключение к администрированию OpenCart

Административная часть магазина OpenCart — это место, где вы можете изменять функции, загружать изображения, добавлять товары, отслеживать клиентов, управлять платежами и многое другое. Настройка в админ-панели влияет на то, как клиент будет взаимодействовать с магазином: изменяя внешний вид, структуру и содержимое витрины магазина.

Доступ к административной панели

Чтобы получить доступ к административной панели, введите адрес магазина в веб-браузере, добавив "/admin" в конце. Например, если ваш магазин находится по адресу "www.chocolatechip.com", ваша админ-панель будет доступна по адресу "www.chocolatechip.com/admin". Даже если магазин расположен в подпапке или на поддомене вашего сайта, добавление "/admin" в конец пути магазина приведет вас к административной части.

Если вы изменили имя папки во время установки, местоположение административной панели изменится на это новое имя. Например, если имя папки администратора изменено на admin1, новое местоположение административной панели будет по адресу "www.chocolatchip.com/admin1".

Изменение имени папки администратора на другое имя важно для повышения безопасности интернет-магазина. Файл config.php необходимо отредактировать, чтобы указать правильный путь к файлу и местоположению.

Имя пользователя и пароль создаются в Шаге 3 автоинсталлятора во время установки. Эти данные теперь можно использовать для заполнения учетных данных для входа администратора.

После ввода правильного имени пользователя и пароля нажатие кнопки "Войти" перенаправит вас на панель управления OpenCart. Когда вы впервые входите в свой магазин, панель управления будет пустой (как показано ниже), потому что еще нет статистических данных для анализа.

Вход в администрирование через панель управления

Панель управления — это первое, что вы увидите при входе в админскую часть OpenCart. Основная функция панели управления — предоставить владельцу магазина обзор работы магазина. В панели управления есть 3 раздела, которые могут помочь вам понять статистические данные, собранные вашим магазином:

  • Обзор:

    Здесь представлены четыре диаграммы, показывающие состояние вашего веб-сайта: общее количество заказов, общие продажи, общее количество клиентов и количество посетителей онлайн, которые помогут вам легче понять аналитику.

  • Мировая карта:

    Мировая карта показывает, откуда поступают заказы с вашего веб-сайта.

  • Аналитика продаж:

    Предоставляется график для отслеживания хронологического прогресса магазина относительно количества заказов и клиентов с течением времени. Значение по оси x — это время, которое может быть в часах, днях или месяцах в зависимости от выбранного диапазона. Значение по оси y показывает общее количество заказов (желтый) и общее количество клиентов (синий).

  • Недавняя активность:

    Раздел для проверки недавней активности любого клиента в вашем магазине, такой как вход в систему, создание учетной записи или размещение новых заказов.

  • Последние заказы:

    Список, который отображает последние заказы и их детали ("ID заказа", "Клиент", "Статус", "Дата добавления", "Итого", и "Действие").

В верхнем меню над панелью управления находится навигация администратора. Вы можете перемещаться между разделами "Каталог", "Расширения", "Продажи", "Система", "Отчеты" и "Помощь". Эти разделы будут подробно объяснены в следующих разделах Руководства пользователя.
