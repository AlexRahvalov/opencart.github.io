用戶

用戶是指登入管理者介面並可以進行修改前台後台的設定的角色。而每個用戶都擁有自己的用戶管，透過用戶群，使用者可以為不同的用戶設置不同的權限去管理網店。

完成了用戶群組的設置後，可以前往同一個分類中的用戶功能頁面。 點擊  建立  會前往上圖的用戶清單。除了基本的用戶信息之外，還必須為用戶創建一個密碼才能登入管理者介面。從下拉式表單中選擇的用戶組群將為用戶分配特定權限。請參閱用戶群組了解相關設定方法。

Username * 用戶登入名字
User Group 用戶群組
First Name * 用戶名字
Last Name * 用戶姓氏
E-Mail * 用戶電郵，不能重覆登記
Image 用戶頭像，透過圖片管理選擇相關圖片
Password * 設置密碼
Confirm * 重覆輸入密碼
Status 設定用戶能否登入管理者介面

無效的訪問權限

The user can access the dashboard the same way the administrator would, with the exception of entering their specific username and password. See Admin interface for more information.
