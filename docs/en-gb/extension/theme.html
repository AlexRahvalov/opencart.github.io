Store Themes

Installable the themes

Theme was grouped into extensions list starting from OpenCart 2 (version 2.3). Each theme can be edited product dimension, product display limited and description lenght. User can go to OpenCart Official Marketplace to buy more themes to stylize the store.

Install & Enable the themes

Install method is same as extension, while download the xxxx_theme.ocmod.zip, go to installer and upload the zip file.

Location Module / Item
Extensions Installer Upload the xxxx_theme.ocmod.zip in installer page

After upload the file successfully, go to Extension select Themes on the dropdown list.

Location Module / Item
Extensions Extensions Themes Select Themes on the dropdown list

New theme will be displayed on the list. Click the install button and can be start to edit the theme, select the directory and edit the setting of the details.
